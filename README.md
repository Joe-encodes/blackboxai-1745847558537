# Netflix Clone

## Project Overview
This project is a front-end implementation of a Netflix-like web application that allows users to explore movies and TV shows available for streaming. The user interface is designed to mirror the popular streaming service, featuring sections for trending content, membership options, and user FAQs.

## Installation
To run this project locally, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   ```

2. **Navigate to the project directory**:
   ```bash
   cd netflix-clone
   ```

3. **Open `index.html` in a web browser** to view the application. There is no additional installation required for this static HTML project.

## Usage
- Open `index.html` in your preferred web browser.
- The main page will load, displaying the hero section with membership options, trending shows, and FAQs.
- Users can enter their email in the sign-up form to get started with the membership.

## Features
- Responsive design that adapts to various screen sizes.
- Displays a hero section with a promotional image and membership call-to-action.
- Lists trending movies and TV shows.
- FAQs section to provide users with information.
- Custom scrollbar for FAQ sections.
- Utilizes Tailwind CSS for styling and layout.

## Dependencies
This project relies on the following external resources:
- **Google Fonts**: Roboto font for typography.
- **Font Awesome**: For icons used in various UI elements.
- **Tailwind CSS**: A utility-first CSS framework for styling.

These resources are included via CDN links in the `index.html` file.

## Project Structure
The project contains the following files:

```
/
├── index.html          # Main HTML file for the application.
```

Inside the `index.html` file, you will find:
- **Header**: Contains the Netflix logo.
- **Hero Section**: Main promotional area for membership enrollment.
- **Pricing Section**: Information about membership costs.
- **Trending Section**: Lists currently trending shows and movies.
- **More Reasons Section**: Highlights features of the Netflix service.
- **FAQ Section**: Provides answers to commonly asked questions.
- **Footer**: Contains additional information about privacy and cookies.

## Conclusion
This Netflix clone project demonstrates the essential concepts of front-end web development using HTML, CSS, and external libraries. Feel free to use this project as a learning tool or a foundation for your own projects.
